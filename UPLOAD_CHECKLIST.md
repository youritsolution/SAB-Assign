# Intune Bulk Assigner Public Hub Checklist

Use this repository as the public-facing hub for Intune Bulk Assigner without publishing the source code.

## Repository Purpose

- public app information
- release notes
- bug reports
- feature requests
- discussions

## Before Publishing

- confirm the website pages are up to date
- confirm the current support and privacy pages reflect the real release state
- confirm the guide matches the shipped build closely enough
- confirm any public download or App Store availability text is accurate

## GitHub Settings

- enable **Issues**
- enable **Discussions**
- add the website URL:
  - `https://youritsolution.dk/intunebulkassigner.html`
- optionally pin a welcome discussion or announcement thread

## Release Notes

For each public release, include:

- version number
- what changed
- key fixes
- known limitations
- support link

## Website Links

The website should point users to:

- app page
- support page
- privacy page
- guide
- discussions
- issues
- releases

## Support Model

- Discussions = questions, ideas, workflow conversation
- Issues = bugs and feature requests
- Email = fallback support path
